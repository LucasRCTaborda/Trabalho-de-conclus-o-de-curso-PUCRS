\begin{resumo}[\protect\bfseries Resumo]
  Este memorial descreve a trajetória acadêmica e prática de Lucas Rubens Correia Taborda
  ao longo de sua atuação na Agência Experimental de Engenharia de Software (\ac{ages}), da
  Pontifícia Universidade Católica do Rio Grande do Sul (\ac{pucrs}), contemplando os períodos
  de \ac{ages} I (2024/1) e \ac{ages} II (2026/1). O documento apresenta os projetos desenvolvidos,
  as atividades desempenhadas em cada sprint, as tecnologias utilizadas e as lições
  aprendidas ao longo da jornada. Em \ac{ages} I, o aluno integrou o projeto ENSportive,
  sistema web para gestão de uma escola de raquetes, utilizando Angular 17, Java Spring Boot
  e PostgreSQL. Em \ac{ages} II, passou a contribuir no projeto Amores Fati, plataforma web
  voltada à inclusão de pessoas com deficiência no mercado de trabalho em tecnologia.
  O memorial evidencia a evolução técnica e comportamental do aluno, destacando o
  crescimento em trabalho em equipe, metodologias ágeis e desenvolvimento de software.
  \bigbreak\
  \\\textbf{PALAVRAS-CHAVE:}
  \ac{ages}. Engenharia de Software. ENSportive. Amores Fati. Desenvolvimento Web. Metodologias Ágeis.
\end{resumo}
